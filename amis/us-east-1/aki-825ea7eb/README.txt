This is a list of all AMIs in us-east-1 that are using this 
particular AKI, which is pvgrub-based.  pvgrub-based AMIs
are much more likely to be cleanly imported as EMIs.

Below this, AMIs are split into sensible buckets.  

"ubuntu" is a list of all AMIs that have "ubuntu" in the name.

"bitmami" is all AMIs that have "bitnami" in the name and 
are instance-store, rather than EBS.
